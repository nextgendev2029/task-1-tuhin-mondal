Programs

What You Will Learn

Six focused modules covering everything from your first HTML tag to deploying a professional portfolio.

📄

HTML Foundations

Learn semantic HTML5, document structure, forms, accessibility best practices, and how to create well-organized, meaningful web pages.

🎨

CSS Responsive Design

Master Flexbox, CSS Grid, media queries, and mobile-first design. Build layouts that look polished on every screen size.

⚡

JavaScript Basics

Understand variables, functions, DOM manipulation, and event handling. Add interactivity and dynamic behavior to your web pages.

✏️

UI/UX Principles

Learn color theory, typography, spacing, and visual hierarchy. Design interfaces that are beautiful, usable, and accessible.

🚀

Portfolio Projects

Build five real projects — landing pages, dashboards, and interactive apps — that demonstrate your skills to employers and clients.

🎯

Career Readiness

Prepare your GitHub profile, write a developer resume, and learn how to present your work in interviews and freelance proposals.

Roadmap

Your Learning Journey

Follow a clear four-step path from beginner to portfolio-ready developer.

1

Learn the Basics

Start with HTML structure, CSS styling fundamentals, and how the browser renders web pages. Build your first static web page by the end of week one.

2

Build Responsive Layouts

Dive into Flexbox, CSS Grid, and media queries. Create layouts that adapt beautifully to mobile, tablet, and desktop screens.

3

Add Interactivity

Learn JavaScript to handle user events, validate forms, manipulate the DOM, and create dynamic page elements that respond to user actions.

4

Publish Your Portfolio

Combine everything into a polished portfolio website. Deploy it on GitHub Pages, set up a custom domain, and share your work with the world.
